CurseForge 앱에서 모드팩을 설치하고, 보이스챗을 확인한 뒤, 화이트리스트 등록을 받고 서버 주소로 접속하세요.

5분 체크리스트

접속 전 이것만 확인하세요

1

마인크래프트 Java 보유

Bedrock이 아니라 PC용 Java Edition이어야 합니다.

2

CurseForge 앱 설치

앱 안에서 모드팩을 검색하고 설치합니다.

3

화이트리스트 등록

디스코드 #화이트리스트 채널에 마크 닉네임을 남겨주세요.

4

서버 주소 입력

멀티플레이에서 34.47.112.1:25565로 접속합니다.

설치

CurseForge 앱에서 모드팩 설치

수동으로 파일을 넣지 말고, 앱에서 같은 모드팩을 설치하는 방식이 가장 쉽습니다.

1. CurseForge 앱 다운로드

공식 다운로드 페이지에서 Windows용 앱을 설치합니다. 설치 후 앱을 켜고 Minecraft를 선택하세요.

2. 모드팩 검색

Minecraft 화면에서 Browse 또는 Search를 누르고 Society: Sunlit Valley를 검색합니다.

3. Install 클릭

제작자가 Chakyl인 모드팩을 고르고 Install을 누릅니다. 서버 기준은 4.0.9 / Minecraft 1.20.1 / Forge입니다.

4. 메모리 설정

프로필 오른쪽의 점 세 개 또는 설정에서 Allocated Memory를 약 8192 MB로 맞춥니다. 멀티 서버여도 내 PC가 모드와 화면을 직접 불러오기 때문에 필요합니다.

5. Play 실행

CurseForge의 Play 버튼을 누르면 Minecraft Launcher가 열립니다. 런처에서 다시 Play를 눌러 게임을 켜세요.

보이스챗

Simple Voice Chat 확인

서버는 근처 사람끼리 말이 들리는 보이스챗을 씁니다. 포트는 서버에서 이미 열려 있습니다.

모드가 없다면 추가

CurseForge에서 Sunlit Valley 프로필을 누른 뒤 Add More Content를 누르고 Simple Voice Chat을 검색해서 설치합니다.

Simple Voice Chat 페이지

게임 안에서 V 키

접속 후 V를 누르면 보이스챗 메뉴가 열립니다. 설정 버튼에서 마이크와 스피커 장치를 고르고 마이크 테스트를 해주세요.

처음 세팅 추천

이어폰 또는 헤드셋을 쓰고, 입력 장치가 실제 마이크인지 확인하세요. 말이 계속 새면 Push To Talk로 바꿔서 테스트합니다.

접속

화이트리스트 등록 후 접속

등록되지 않은 닉네임은 서버에서 바로 튕깁니다. 닉네임은 대소문자를 정확히 적어주세요.

  1. 디스코드 #화이트리스트 채널에 마인크래프트 닉네임을 남깁니다.
  2. 관리자가 등록했다는 답을 주면 게임을 켭니다.
  3. Multiplayer를 누릅니다.
  4. Add Server 또는 Direct Connection을 누릅니다.
  5. 주소에 34.47.112.1:25565를 입력하고 접속합니다.
복사용 주소 34.47.112.1:25565

지도와 땅 보호

J 키, 청크 보호, 강제 로딩

자기 집이나 농장은 FTB Chunks로 보호하세요. 서버 기본 제한은 개인당 보호 청크 16개, 강제 로딩 3개입니다.

지도 열기

보통 J 키로 큰 지도를 엽니다. 안 열리면 Options → Controls에서 Map 또는 FTB Chunks를 검색해 키를 확인하세요.

청크 보호하기

인벤토리의 파란 지도 아이콘 또는 FTB Chunks 지도를 열고, 보호할 칸을 클릭합니다. 여러 칸은 클릭한 채 드래그합니다.

보호 해제하기

보호된 칸을 우클릭합니다. 여러 칸은 우클릭한 채 드래그합니다.

강제 로딩

보호한 청크에서 Shift를 누른 채 클릭하면 강제 로딩을 켭니다. 초록 선이 보이면 켜진 상태입니다. 너무 많이 켜면 서버가 느려질 수 있습니다.

자세한 기준은 FTB Chunks 공식 문서 를 참고하세요.

초반 플레이

들어오면 먼저 할 것

퀘스트북 확인

모드팩 진행 방향은 퀘스트북을 따라가면 됩니다. 농사, 동물, 낚시, 광산 순서로 천천히 열립니다.

마을과 야생 구역

허브와 웨이스톤 주변은 공용 구역입니다. 개인 집은 자기 청크를 보호하고 짓습니다.

웨이스톤 사용

허브의 웨이스톤으로 주요 이동 지점에 갈 수 있습니다. 이름이 정리된 곳만 사용하세요.

서버 렉 줄이기

멀리 날아다니며 새 청크를 빠르게 열면 서버가 느려집니다. 새 지역 탐험은 천천히 이동해주세요.

규칙

서버 기본 약속

남의 청크 건드리지 않기 보호가 안 된 곳이어도 다른 사람 건축물은 건드리지 않습니다.
공용 웨이스톤 주변 정리 허브, 야생, 광산 포탈 주변에는 개인 창고나 농장을 만들지 않습니다.
강제 로딩은 필요한 곳만 개인당 3개 제한입니다. 농장이나 기계가 꼭 돌아가야 하는 곳에만 씁니다.
문제 생기면 좌표 공유 렉, 보호 문제, 접속 문제는 좌표와 상황을 같이 알려주세요.

문제 해결

자주 막히는 부분

화이트리스트가 아니라고 떠요

디스코드 #화이트리스트 채널에 적은 닉네임이 실제 마인크래프트 닉네임과 같은지 확인하세요.

보이스챗이 연결 안 돼요

Sunlit Valley 프로필에 Simple Voice Chat이 설치되어 있는지 확인하고, 게임 안에서 V 키를 눌러 장치를 다시 고르세요.

지도가 안 열려요

Controls에서 J 키가 다른 기능과 충돌하는지 확인하세요. Map 또는 FTB Chunks로 검색하면 관련 키가 나옵니다.

렉이 심해요

클라이언트 메모리를 8GB 근처로 올리고, 쉐이더를 끄고, 렌더 거리를 낮춘 뒤 다시 접속하세요.
